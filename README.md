# abacus
## About
Table data editor with the support of formulas

## Features
- cells edition
- graphical UI
- formulas with custom language
- endless scroll
- syntax highlighting and helper
- basic styles and fonts editor

## Formula language
```
123
text
=add(1,2)
=add(A1, B5) vararg
=add(A1:B5) cell ranrange
=add(A:B) column range
=add(1:5) row range
```

## Technologies
- Java 15
- Swing
